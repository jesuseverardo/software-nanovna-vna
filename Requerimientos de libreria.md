**Requisitos e instalación del software NanoVNA**



Para poder ejecutar correctamente el software, es necesario tener instalado Python, de preferencia la versión Python 3.10.10, ya que fue la versión utilizada durante el desarrollo del proyecto.



Además, el programa necesita algunas librerías externas para funcionar correctamente. Estas librerías permiten generar las gráficas, realizar cálculos numéricos y establecer comunicación con el NanoVNA mediante el puerto serial.



Las librerías necesarias son:



1. matplotlib

2\. numpy

3\. pyserial



**Instalación de las librerías**



Para instalarlas, primero se debe abrir una terminal o consola de comandos dentro de la carpeta del proyecto (CMD).



Después, se puede ejecutar el siguiente comando: *pip install matplotlib numpy pyserial*







Una vez instaladas las librerías, el programa se puede ejecutar desde el archivo principal: *principal.py*
